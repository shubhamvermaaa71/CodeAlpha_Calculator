# Glassmorphism Calculator

A modern calculator application with a premium glassmorphism interface, animated visual effects, and smooth user interactions.

## Features

- Basic Arithmetic Operations
- Glassmorphism UI Design
- Animated Background Effects
- Smooth Button Animations
- Responsive Layout
- Keyboard Support
- Real-Time Calculations
- Blur and Transparency Effects
- Clean User Experience

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Supported Operations

- Addition
- Subtraction
- Multiplication
- Division
- Decimal Calculations

## Future Improvements

- Scientific Mode
- Calculation History
- Theme Customization
- Memory Functions
